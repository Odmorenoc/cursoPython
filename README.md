### 3.1. BusquedaOrdenación

Corresponde a programas relacionados con algoritmos de busqueda y ordenación de datos.

__busquedaLineal.py__

Programa que recorre una lista con numeros aleatorios, identificando si el numero buscado se encuentra.

__enumeracionExhaustiva.py__

Es un programa que busca la raiz cuadrada de un numero, mediante el calculo de todas las opciones hasta encontrar una raiz cuadrada igual o menor a la buscada.

__aproximacionSoluciones.py__

Es un programa que devuelve una aproximación de la raiz cuadrada mediante el algoritmo de aproximación exhaustiva o el algoritmo de busqueda binaria.

___busquedaBinariaRecursivo.py__

Implementación del algoritmo de Busqueda Binaria, mediante recursividad, en el que se busca de una lista de numeros aleatorios si un valor se encuentra.

__ordenamientoBurbuja.py__

Corresponde a un algoritmo de ordenación de una lista de numeros de menor a mayor, mediante la comparación de pares ordenados de elementos de la lista, se utiliza recursividad.

__ordenamientoInsercion.py__

Es un programa que ordena una lista de numeros aleatorios mediante la inserción del valor menor en la posición indicada verificando que el valor anterior sea menor al actual.

__ordenamientoMezcla.py__

Es una mezcla entre el ordenamiento por inserción y la busqueda binaria, de manera que se busca ordenar una lista, diviendola losta original en dos partes izquierda y derecha, hasta que en cada sublista solo se encuentre un elemento, entonces se comparan para obtener una sublista ordenada, posteriormente se comparan las dos sublistas para definir la lista ordenada.
